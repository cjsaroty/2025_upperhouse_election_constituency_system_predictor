# 2025年参議院選挙データを活用した選挙方針策定と候補者選定プロジェクト

## プロジェクト概要

2025年参議院選挙の候補者データ（選挙区）を用いて、
**LightGBM による当落予測モデル** を構築したプロジェクトです。

本プロジェクトでは、2025年参議院選挙データを用いて「どのような候補者を、どの選挙区に擁立すべきか」
という意思決定を定量的に支援することを目的に、LightGBMによる当落予測モデルを構築しました。

# 使用した特徴量（Feature Engineering）

有権者の投票選択の4つの基準(①支持政党②政策争点③候補者の属性(①~③はミシガンモデル)④過去の業績(Fiorina 1981))を当落の要因として分類し、
当落に寄与しうるカテゴリ・数値特徴量を設計。

** 特徴量一覧**

| 種類    | 特徴量名                                |
| ----- | ----------------------------------- |
| 支持政党  | 所属政党年齢、性別                               |
| 重要争点に関する立場 | 争点優先度（順位1〜3位） |
| 候補者の属性  | 性別、選挙区、地元出身フラグ 、元・現・新人、地方議会経験、秘書経験                  |
| 過去の業績    | 職業（カテゴリを統合し分類）、過去当選回数（参・衆）                   |

カテゴリ変数には Label Encoding / CatBoost Encoding を使用。

# モデル構築

### ■ 採用モデル

**LightGBM（LGBMClassifier）**

* 学習が速い
* 精度が高くビジネス利用に向く

### ■ ハイパーパラメータ

* 交差検証（KFold）を用いてチューニング
* 主に以下を最適化：

  * num_leaves
  * max_depth
  * learning_rate
  * feature_fraction
  * min_child_samples

# 主な分析結果（サマリー）
*特徴量重要度
-年齢＞争点二位＞所属政党＞争点3位＞争点1位＞議席数＞性別＞職業(分類)＞衆参当選回数＞出生地からの立候補か否か＞政府規模＞地方議会経験の有無＞参議院当選回数＞元現新＞秘書経験の有無

*最も当選しやすい属性
-年齢(40-50代)＞争点2位(地方活性化・人口減)＞所属政党(公明党)＞争点3位(農林水産業)＞争点1位(防災・復興)＞議席数(議席数4)＞性別(女性)＞職業(教育・研究)＞衆参当選回数(4と8)＞出生地からの立候補か否か(出生地からの立候補)＞政府規模(どちらかといえば大きな政府(4))＞地方議会経験の有無(地方議会経験有り)＞参議院当選回数(過去当選回数が多いほど当選しやすいが回数で言うと4回)＞元現新(現)＞秘書経験の有無(秘書経験有り)
