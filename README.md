# 📘 2025年参議院選挙データを用いた LightGBM 当落予測モデル

**〜候補者選定と選挙戦略立案のための機械学習プロジェクト〜**

## 🎯 プロジェクト概要

2025年参議院選挙の候補者データ（選挙区）を用いて、
**LightGBM による当落予測モデル** を構築したプロジェクトです。

選挙データ特有の複雑なカテゴリ変数（政党、元現新、職業など）に対応するため、
前処理・特徴量エンジニアリングに重点を置き、
精度と解釈性を両立したモデル構築を行いました。

# 🧩 使用した特徴量（Feature Engineering）

選挙データのドメイン知識を活かし、
当選確率に寄与しうるカテゴリ・数値特徴量を設計。

**✔ 特徴量一覧**

| 種類    | 特徴量名                                |
| ----- | ----------------------------------- |
| 基本属性  | 年齢、性別                               |
| 政治的属性 | 所属政党、元・現・新人、過去当選回数（参・衆）、地方議会経験、秘書経験 |
| 地域特性  | 選挙区、議席数、地元出身フラグ                     |
| 職業    | 職業（カテゴリを統合し分類）                      |
| 政策指向  | 争点優先度（順位1〜3位）                       |

カテゴリ変数には Label Encoding / CatBoost Encoding を使用し、
LightGBM に最適化しました。

---

# 📊 モデル構築

### ■ 採用モデル

**LightGBM（LGBMClassifier）**

* カテゴリ変数に強い
* 学習が速い
* 選挙データのような複雑な相互作用を捉えられる
* 精度が高くビジネス利用に向く

### ■ ハイパーパラメータ

* 交差検証（KFold）を用いてチューニング
* 主に以下を最適化：

  * num_leaves
  * max_depth
  * learning_rate
  * feature_fraction
  * min_child_samples

---

# 📈 モデル評価

### ■ 評価指標

* Accuracy（正解率）
* Precision（適合率）
* Recall（再現率）
* F1-score
* 混同行列

LightGBM が最も高いバランスの取れたスコアを示し、
ロジスティック回帰や他の木系モデルよりも高精度でした。

### ■ 重要度分析

LightGBM の feature importance により、
当落に寄与する因子を定量的に可視化。

上位は以下の通り：

1. **元・現・新人**
2. **政党**
3. **年齢**
4. **職業**
5. **議席数**

政治的経験・組織力が強く影響し、
年齢・職業などの個人属性も寄与していました。

---

# 🔍 主な分析結果（サマリー）

* **現職＞与党・主要野党＞40–60代＞教育・研究職** が当選しやすい層
* 選挙区の議席数は 1〜4 議席区で安定し、7議席（東京）は競争が激しい
* 性別は当選率に大きな差が見られない
* 元現新・政党・年齢が特に重要な説明変数であることが LightGBM によって確認された
